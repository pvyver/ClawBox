# ClawBox 🦞

Your AI lobster companion's dashboard — monitoring health, cron jobs, and token usage on a Jetson-powered ClawBox.

## Sections

- **Dashboard** — Overview with live snapshot cards
- **Health** — CPU, temperature, memory, disk, and GPU metrics
- **Cron Jobs** — Scheduled tasks and job registry
- **Token Usage** — Per-model consumption, budgets, and alerts

## Tech

- Jekyll static site, hosted via GitHub Pages
- Dark theme with ClawBox orange accent (#f97316)
- Auto-updated by ClawBox background jobs
